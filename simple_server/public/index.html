静态文件由 package:shelf_static 从 /public 文件夹中获取。

这个 Web Server 由 Student 20181060 使用 package:shelf_router 定义了以下一系列的 API:

  • /helloworld
  • /time
  • /sum/<a|[0-9]+>/<b|[0-9]+>
  • /any-string/<[A-Za-z]+>

最后一个 Handler Request 接受以下格式的 URL 并获取参数，如： /sum/1/2 和 /sum/35/7, 但不能是 /sum/bob/john 或 /sum/1/2/3格式。
